This is an experiment / work-in-progress. Nothing to see here (yet).

## Prerequisites

The following must be installed prior to building this project:

* JDK >= 11
* Maven >= 3.6.3
* Docker >= 20.10.1

## Build the Project

To build the Docker image that's produced by this project, run:

    mvn verify
